What's New in 1.11

In/out point trimming, GIF export, watermark overlays, frame-accurate navigation, file size estimation, and updater error diagnostics.

  • New: In/Out Point Trimming (Pro) — Mark sub-ranges with draggable handles, keyboard shortcuts (I/O/X), and timecode input. Works across all pipelines. CLI: --in / --out
  • New: Watermark Overlay (Pro) — Burn a transparent PNG watermark with scale, opacity, and drag-to-position. Live preview shows exact output placement
  • New: GIF Export — Animated GIF output with configurable frame rate and loop count. --codec gif on CLI
  • New: Frame-Accurate Navigation — Left/right arrow keys nudge the playhead by exactly one frame
  • New: File Size Estimation — Redesigned Automatic quality mode targets 70% of source bitrate. New estimation table shows Source vs. Output with size comparison
  • New: No Audio Option — Strip all audio tracks from the output
  • New: Sources Size Column — Optional file size column in the Sources panel, sortable by size
  • New: Preset Search — Search bar in the Presets panel to filter by name
  • Improved: Update Error Details — When an auto-update fails, a follow-up alert now shows the specific reason (bad archive, signature failure, permission denied, etc.)
  • Improved: About Window — Credits now shown in the About panel
  • Fixed: Overlay positioning and clipping on non-16:9 and rotated sources
  • Fixed: Backspace no longer removes sources when a text field is focused
