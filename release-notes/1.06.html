What's New in 1.06

  • New: Command-Line Interface (Pro) — pme CLI tool for headless transcoding, file probing, preset management, and license activation. Supports all codecs, presets, LUTs, and overlays. JSON output mode for scripting and automation. CLI docs
  • New: Service Mode (Pro) — run PME as a background daemon with a local REST API. Submit jobs, monitor progress, and control the queue over HTTP. Supports Unix domain sockets, TCP localhost, SSE progress streaming, bearer token auth, and launchd auto-start
  • New: AppleScript Support — automate transcoding with transcode, probe, and list presets commands. Works with Script Editor, Automator, and Folder Actions for watch-folder workflows. AppleScript docs
  • New: Shortcuts / App Intents — three Shortcuts actions (Transcode, Probe, List Presets) with Siri phrase support
  • New: Install CLI Tool — menu item creates a /usr/local/bin/pme symlink for terminal access
  • Improved: License activation now shows a thank-you message and displays the registered name and email from your Gumroad purchase
