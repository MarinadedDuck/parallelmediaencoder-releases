What's New in 1.09

Watch folders for automatic transcoding — drop media into a folder and PME picks it up, applies a preset, and transcodes to your output location in the background.

  • New: Watch Folders (Pro) — Configure folders to monitor for new media. PME polls at a configurable interval (1–60 min), moves files to a PickedUp subfolder, and auto-queues them with your chosen preset and output location
  • New: Watch Folders Window — Dedicated management window with add/edit/remove, start/stop per folder, Scan Now button, and picked-up file counter
  • New: CLI Watch Commands (Pro) — pme watch add/list/start/stop/remove for terminal-based watch folder management. Shared config with the GUI
  • New: Network Folder Support — Watch folders work with SMB, NFS, and AFP mounted volumes via polling
  • New: Delete Key Source Removal — Select sources and press Delete/Backspace to remove them
  • New: Undo/Redo — Cmd+Z to undo removed sources, Cmd+Shift+Z to redo. Works for single, multi-select, and "Remove Unsupported"
  • Improved: Queue Integration — Watch folder jobs share the main queue with a green "Watch Folder" badge. Status bar shows active watch folder count
  • Improved: Shutdown Guard — Warning dialog when quitting with active watch folders or encodes
