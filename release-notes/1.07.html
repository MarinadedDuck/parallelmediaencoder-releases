What's New in 1.07

Full vertical video support with rotation controls, a built-in metadata viewer, and resize modes (Stretch, Fit, Crop) for custom resolutions.

  • New: Video Rotation — Rotate clips 90° CW/CCW with preview thumbnails. Works across all formats (MOV, MXF, BRAW, R3D, MKV). Batch rotation, CLI --rotate flag, and preset support included
  • New: Resize Modes — Choose Stretch, Fit (letterbox), or Crop when using custom resolutions with mismatched aspect ratios. Visual side-by-side previews show each mode's effect
  • New: Override Settings — Control parallel encode limits, GPU decode slots, and rotation memory limits from a new Settings tab
  • New: Session-Sticky Output Folder — Picking an output folder now applies it to all unconfigured clips and future imports for the session
  • New: Vertical/Portrait Video — Full orientation support for iPhone and portrait-shot video with correct dimension handling
  • New: Batch Custom Resolution — Apply arbitrary custom resolutions with resize modes in batch mode
  • New: Keep Aspect Ratio Toggle — Shows the source ratio (16:9, 9:16, 17:9, etc.) and snaps to AR-preserving values
  • Improved: PME Metadata — All output files carry encode parameters; inspector shows a dedicated PME metadata section
  • Improved: Memory Management — Rotation encodes use backpressure, reduced concurrency, and GPU resource reclamation to stay within system RAM
  • Fixed: Multiple rotation pipeline bugs (squished BRAW/R3D, missing transforms in DNx path, operator precedence)
  • Fixed: Portrait video orientation, resize mode persistence, batch settings sync
