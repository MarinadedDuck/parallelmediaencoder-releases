What's New in 1.04

  • New: Audio preview playback — hear audio while previewing video files (AVFoundation, MXF, MKV/WebM, BRAW, R3D)
  • New: Audio-only file preview — mp3, wav, flac, and other audio files now show a waveform view with playback controls
  • New: Volume slider and mute toggle in preview controls (persists across sessions)
  • New: Pause/Resume queue — truly suspends active encodes mid-frame and resumes from the same position
  • New: Toolbar shows icons with text labels, Add to Queue has Enter keyboard shortcut
  • New: Encode menu with Add to Queue shortcut
  • Fixed: Frame rate upconversion now works correctly for MP4/MOV files (was only fixed for MXF in v1.02)
  • Fixed: Window close during encoding no longer causes looping quit dialog or crash — app stays in dock, click to restore
  • Fixed: Partial output files cleaned up on cancel/quit (including AVAssetWriter journal files)
  • Fixed: Queue clear no longer leaves orphaned job in settings panel
  • Fixed: Cancel encoding shows confirmation dialog before cancelling
  • Fixed: Insta360 .insv/.lrv transcoding crash with missing AVChannelLayoutKey
  • Fixed: HEVC Main10 profile auto-downgrades to Main for 8-bit sources (prevents purple/white color corruption)
  • Fixed: BRAW framework embedding no longer fails on incremental builds
  • Changed: "Cost" terminology replaced with "Slots" in queue UI
