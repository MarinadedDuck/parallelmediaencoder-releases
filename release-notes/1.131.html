What's New in 1.131

Edit-friendly H.265 presets, codec hardware limit lockouts, expanded keyframe controls, and ProRes RAW preview improvements.

  • New: Edit-Friendly H.265 Presets — Two new Video Editing presets that produce HEVC files with all-intra encoding for instant scrubbing in Final Cut Pro and Premiere Pro. 10-bit (Main10 4:2:2, 150 Mbps) and 8-bit (Main, 100 Mbps) variants, both with PCM 24-bit audio passthrough
  • New: All Intra Keyframe Option — Every frame is an I-frame for maximum seek performance. Also added 2-frame GOP matching Atomos Shogun's I-P pattern
  • New: Codec Hardware Limit Validation — The app now prevents queuing jobs that exceed your hardware encoder's limits. H.264 is validated at 4096x2160 @ 60fps, HEVC at 8192x4320 @ 120fps, and ProRes at 8192x4320 @ 120fps. Orange warnings appear when source resolution or frame rate exceeds the selected codec's maximum, and the Add to Queue button is disabled until the issue is resolved
  • Improved: ProRes RAW Preview — Faster scrubbing with cached preview sessions, hybrid seek (fast approximate drag, exact frame on release), 1080p preview cap for smoother playback, and instant Flat/Rec.709 mode switching
  • Fixed: ProRes RAW Rec.709 Colour — Preview and export now use the same colour pipeline. The Rec.709 LUT is locked to the correct transform and bundled in the app, preventing silent look drift
  • Improved: Resize Preview Thumbnails — Rotation and Stretch/Fit/Crop preview thumbnails are now capped so they don't grow excessively large when switching between landscape and portrait orientations

What's New in 1.13

ProRes RAW colour profiles, native ProRes-in-MXF decode, major queue and preset UI improvements, and critical bug fixes for watermark/trim export, DNxHD output, and queue selection.

  • New: DNxHD CLI Codec Mappings — pme transcode --codec dnxhd-lb/sq/hq/hqx/444 now available alongside the existing DNxHR variants
  • New: Job Status Banners — The settings panel shows a live-updating banner for queued, encoding, and completed jobs. Completed jobs have a clickable banner that opens the output file in Finder
  • New: Reset Status — Right-click completed, failed, or cancelled jobs in the queue to re-run them
  • New: Encoder Stats in Queue — Optional "Show encoder FPS and realtime rate" toggle in Settings displays live encoding speed on running jobs
  • New: Preset Detail View — Expand any preset with the chevron icon to see all saved settings at a glance
  • Improved: "Applied" Preset Feedback — The Apply Preset button shows "Applied" when the selected preset matches the current job settings, and reverts when you make changes
  • Improved: Reactive Settings Editability — Settings lock and unlock in real time as jobs transition between queued, encoding, and completed states
  • Fixed: Watermark Export Regression — Watermarks now appear in encoded output when "Keep sources in staging" is enabled. Previously, watermark settings were lost when jobs entered the queue. Trim ranges, GIF loop count, and several other settings were also affected and are now fixed
  • Fixed: DNxHD Output — DNxHD codec variants now correctly produce DNxHD streams instead of silently falling back to DNxHR
  • Fixed: Queue Selection Highlight — Row highlights in the queue panel no longer freeze after encoding finishes
  • Fixed: Preset Tag on No-Audio Sources — Preset names no longer disappear on import for sources without audio tracks
