What's New in 1.12

Camera RAW decode controls, LucidLink presets, 10-bit SDR encoding fixes, and encoding settings improvements.

  • New: Camera RAW Controls (R3D + BRAW) — Floating always-on-top window for overriding RAW decode settings. Switch between "As Shot" (camera metadata) and "Custom" to adjust color science, color space, gamma, ISO, white balance, and exposure with real-time preview updates
  • New: RED R3D Controls — Color science (Legacy/IPP2), color space (REDWideGamutRGB, Rec.709, Rec.2020, DCI-P3, and more), gamma curve (Log3G10, Log3G12, REDlogFilm, BT.1886, HLG, and more), ISO, and drag-to-scrub exposure control
  • New: Blackmagic RAW Controls — Color space (BMD, Rec.709, Rec.2020, DCI-P3), gamma (BMD Film/Video/Extended Video, Rec.709), white balance, ISO, and exposure
  • New: LucidLink Presets — Three presets for LucidLink cloud workflows: Normal (HEVC Main10, 21 Mbps), Proxy (HEVC Main, 14 Mbps, half res), High (HEVC Main10 4:2:2, 26 Mbps)
  • New: Encoding Profile Warnings — Yellow warnings when selecting Main10 or 4:2:2 profiles on sources that don't benefit from them
  • New: Info Tooltips — Hover info icons on Automatic quality, Profile, Keyframe Interval, and Scaled resolution settings
  • Fixed: 10-bit SDR encoding — Sources like Canon XF-HEVC (10-bit 4:2:2) are no longer downgraded to 8-bit. Correct color metadata tags (bt709) now written on all HEVC/H.264 outputs
  • Fixed: hvcC bit depth parsing — HEVC Rext sources now correctly report bit depth and chroma format in metadata
  • Improved: Automatic bitrate — HEVC targets 50% of source bitrate (was 70%), H.264 targets 65%
  • Improved: H.264 Baseline — B-Frames toggle grayed out when Baseline profile is selected
