What's New in 1.03

  • Fixed: Blackmagic RAW runtime is now embedded in the app — BRAW files no longer require a separate Blackmagic RAW SDK or Player install
  • Fixed: RED R3D decoding now falls back to OpenCL or CPU when Metal GPU is unavailable (e.g. virtual machines)
  • Fixed: Improved error messages when RAW codec initialization fails
  • Fixed: Settings window and console now render correctly on Light Mode systems
  • New: Welcome screen on first launch lets you enter a license key right away
